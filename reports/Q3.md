# 问题3 报告：压缩/格式/几何变换鲁棒性

- 结论：空域 LSB/LSBM 对有损压缩与几何变换基本不鲁棒，原图可恢复，JPEG 与几何/模糊后大多失败。

## 实验结果
| 场景 | 提取成功 | 恢复长度 |
|---|---|---|
| orig | 否 | 30 |
| jpeg95 | 否 | 486776 |
| jpeg85 | 否 | 486817 |
| jpeg75 | 否 | 487614 |
| resize0.75 | 否 | 487174 |
| rotate2 | 否 | 494281 |
| blur1 | 否 | 486799 |

## 改进思路
- 改为 JPEG 频域隐写（J-UNIWARD/UED/UERD 等）。
- 引入纠删编码、重复扩频与同步标记，缓解位翻转与丢失。
- 自适应嵌入：边缘/纹理区优先，降低可检性与破坏风险。
